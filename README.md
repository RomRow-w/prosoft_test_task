  <h3>Установка:</h3>
  <p>В корне проекта вызываем комманду:</p>
  <pre>npm install</pre>
  <h3>Запуск</h3>
  <p>После установки в корне проекта вызываем комманду:</p>
  <pre>npm start</pre>
  <p>После чего проект откроется в вашем браузере по адресу: http://localhost:3000/</p>
